# 📊 Netflix EDA Project

## 📌 Objective
Analyze Netflix's content data to discover trends, patterns, and insights using Python and Pandas.

## 📁 Dataset
- Source: [Netflix Shows Dataset on Kaggle](https://www.kaggle.com/datasets/shivamb/netflix-shows)
- Format: CSV

## 🧰 Tools Used
- Python
- Pandas
- Matplotlib & Seaborn
- Jupyter Notebook
- WordCloud (for text visuals)

## 🔍 Key Questions Answered
- Movies vs TV Shows: What's more on Netflix?
- Which year had the most content added?
- Top countries contributing to Netflix content
- Most popular genres
- Content trends over time

## 📈 Visualizations
- Bar plots, histograms, word clouds
- Grouped bar chart (Movies vs TV Shows by year)
- Genre popularity and country-wise content

## 📎 Files Included
- `netflix_eda.ipynb` — Main notebook with code and analysis
- `data/netflix_titles.csv` — Raw data file
- `images/` — All saved charts and plots
- `README.md` — This file

## 📌 Summary
This project showcases the ability to perform data cleaning, feature engineering, visualization, and storytelling using a real-world dataset.
